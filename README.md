## KÄYTTÖOHJE

Asenna sovellus.

Käynnistys komentoriviltä komennolla:
- `./gradlew run --console=plain` 


Ohjelman komennot:
- `help` - listaa kaikki komennot
- `uusi` - lisää uusi vinkki
- `sulje` - sulje sovellus
